🔐 HOW HASHING WORKS
Each file's bytes are processed by the browser's native Web Crypto API using the SHA-256 algorithm.
The resulting 64-character hex digest is unique to that exact file. Any modification, even one byte, produces a completely different hash — making evidence tamper-proof.
